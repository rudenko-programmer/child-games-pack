# Документация по игре "Абетка"

## Назначение

"Абетка" - обучающая игра для детей, предназначенная для изучения украинских букв через звук, визуальные ассоциации и короткие игровые сессии.

Целевая аудитория: дети дошкольного и младшего школьного возраста.

## Режимы игры

### Пазлы (`PuzzleGame`)

Сценарий:
1. Игрок выбирает режим букв (гласные, согласные, смешанный) и сложность (количество вариантов ответа).
2. Игра произносит целевую букву.
3. Игрок выбирает вариант.
4. При правильном ответе открывается часть изображения пазла.
5. После полного открытия пазла уровень завершается, доступен переход на следующий.

Ключевая логика:
- прогресс уровней: 1..20;
- увеличение плотности сетки пазла с ростом уровня;
- анти-повторы последних целевых букв;
- сохранение прогресса в `localStorage`.

### Бульбашки (`BubbleGame`)

Сценарий:
1. Игрок выбирает набор букв и скорость движения пузырей.
2. Игра задает целевую букву.
3. Появляются пузыри с буквами.
4. Игрок нажимает правильные пузыри, получая очки.

Ключевая логика:
- адаптивные интервалы спавна и длительность анимации по скорости;
- шанс появления целевой буквы среди обычных;
- ограничение частых повторов одинаковых букв;
- штрафная анимация для ошибочного выбора (пузырь-камень).

## Озвучка и работа с аудио

Файл: `src/data/letters.ts`

Реализовано:
- воспроизведение mp3 для букв;
- воспроизведение случайных фраз похвалы/ошибки;
- остановка предыдущего аудио перед новым;
- fallback на `SpeechSynthesis` (`uk-UA`) при проблемах с mp3.

Примечание для Electron:
- используются относительные пути `./audio/...` для корректной работы в desktop-сборке.

## Сохранение прогресса

Используется `localStorage` (ключ `ABETKA_GAME_PROGRESS`).

Сохраняемые поля:
- `level`
- `score`
- `revealedPieces`
- `mode`
- `difficulty`
- `maxLevelReached`

## Ресурсы проекта

### Изображения пазлов
- Папка: `public/images/puzzles/`
- Формат: `.jpg`
- Именование: обычно `1.jpg` ... `20.jpg`

### Аудио
- Папка: `public/audio/`
- Буквы: `а.mp3`, `б.mp3` и т.д.
- Поощрения: `praise_0.mp3 ... praise_20.mp3`
- Ошибки: `error_0.mp3 ... error_9.mp3`

### Скрипты подготовки контента
- `scripts/download_puzzles.js` - загрузка изображений пазлов.
- `scripts/generate_audio.js` - загрузка и генерация аудио через TTS-источник.
