# Child Games Pack - "Абетка"

Интерактивная детская игра для изучения украинского алфавита с озвучкой и визуальными мини-играми.

Проект включает два игровых режима:
- `Пазлы`: угадывание букв с открытием частей картинки.
- `Бульбашки`: поиск нужной буквы среди всплывающих пузырей.

Игра работает как веб-приложение, PWA и как desktop-приложение через Electron.

## Что внутри

- Изучение букв украинского алфавита (гласные, согласные, смешанный режим).
- Озвучка букв и поощрительных/корректирующих фраз.
- Прогрессия уровней в пазлах (до 20 уровней).
- Сохранение прогресса пазлов в `localStorage`.
- Эффекты победы (`canvas-confetti`).

## Игровые механики

### 1) Пазлы

- Выбор режима обучения: гласные/согласные/все буквы.
- Выбор сложности: количество вариантов ответа (`2`, `4`, `6`, `8`).
- При правильном ответе:
  - увеличивается счет;
  - открывается случайный фрагмент пазла;
  - воспроизводится похвала.
- При ошибке:
  - отмечается неправильный вариант;
  - может скрыться один уже открытый фрагмент (не больше одного штрафа за раунд);
  - воспроизводится корректирующая фраза.
- После открытия всех фрагментов уровня запускаются фейерверки и доступна кнопка перехода к следующему уровню.

### 2) Бульбашки

- Перед стартом выбираются:
  - набор букв (гласные/согласные/все);
  - скорость (`slow`, `normal`, `fast`).
- Игра произносит целевую букву, игрок лопает нужные пузыри.
- За правильный пузырь начисляются очки и выбирается новая цель.
- Неправильный пузырь превращается в "камень" и падает.

## Архитектура проекта

- `src/App.tsx` - навигация между главным меню и мини-играми.
- `src/components/MainMenu.tsx` - главное меню.
- `src/components/PuzzleGame.tsx` - режим пазлов, логика уровней и прогресса.
- `src/components/BubbleGame.tsx` - режим пузырей.
- `src/data/letters.ts` - наборы букв, озвучка, fallback на Speech API.
- `electron/main.cjs` - окно Electron и запуск desktop-версии.
- `electron/preload.cjs` - preload-скрипт (пока заглушка).
- `scripts/generate_audio.js` - генерация/загрузка озвучки.
- `scripts/download_puzzles.js` - загрузка изображений для пазлов.

## Требования

- Node.js 18+ (рекомендуется актуальная LTS-версия).
- npm (идет вместе с Node.js).

## Запуск и сборка

Установка зависимостей:

```bash
npm install
```

Запуск в режиме разработки (web):

```bash
npm run dev
```

Сборка web-версии:

```bash
npm run build
```

Предпросмотр production-сборки:

```bash
npm run preview
```

Запуск Electron в разработке:

```bash
npm run electron:dev
```

Сборка Electron-пакета:

```bash
npm run electron:build
```

## Аудио и изображения

Ожидаемые ресурсы:
- `public/audio/*.mp3` - буквы и служебные звуки (`praise_*.mp3`, `error_*.mp3`).
- `public/images/puzzles/*.jpg` - изображения уровней (обычно `1.jpg` ... `20.jpg`).

Важно для Electron: используются относительные пути вида `./audio/...` и `./images/...`.

Если mp3 недоступен, используется fallback через `SpeechSynthesis` (`uk-UA`).

## PWA

Проект настроен через `vite-plugin-pwa`:
- автообновление service worker;
- манифест приложения;
- кеширование статических ассетов.

## Хранение прогресса

Пазлы сохраняют состояние в `localStorage` по ключу:

- `ABETKA_GAME_PROGRESS`

Сохраняются уровень, очки, открытые фрагменты, выбранный режим и сложность.
